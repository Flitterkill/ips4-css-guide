Grid Framework

IPB consists of a 12-column grid system, which is relatively easy to grasp. The spans can be swapped around in any order and where the spams sum to more than 12, the overflow will move to a new line.

Please note: For the grid to take effect you must apply dir='rtl' to the html tag.

An example of the grid system in action:
