Buttons

In IPS 4.x the system provides a method to create a new theme by duplicating the defualt IPS theme. As part of this a number of defualt end-user theme settings are provided for changing items such as button colors, background, and so on. Conflicts can occur with theme importing if a theme designer/editor hard codes CSS variables that have a custom theme setting without removing that setting from the theme. For example, if you add a css file to a theme or use custom.css and set the body element to have a background-color: blue; this will function perfectly well on your developing instance no matter if you are in or out of designers mode or easy mode.
